دستورات برای Push به GitHub:

1. بررسی وضعیت:
git status

2. افزودن فایل‌ها:
git add app.py
git add Dockerfile

یا همه فایل‌ها:
git add .

3. Commit:
git commit -m "اضافه کردن فایل ساده Python و Dockerfile"

4. Push:
git push origin main

اگر branch شما master است:
git push origin master

اگر remote تنظیم نشده:
git remote add origin https://github.com/USERNAME/REPO.git
git push -u origin main
